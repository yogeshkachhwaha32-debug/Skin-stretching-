# skin stretching simulation 🚀

Anime Power Simulator is a real-time computer vision application that captures your webcam stream, tracks facial landmarks and hand gestures using MediaPipe, and overlays dynamic anime-inspired visual effects. The app features procedural audio synthesis and live MP4 recording.

---
## Current Features

### Luffy (Gum Gum Stretch)

Inspired by Monkey D. Luffy from One Piece.

- Uses a webcam to track the user's face in real time.
- Detects mouth opening and facial landmarks.
- Dynamically stretches the mouth region based on mouth movement.
- Elastic deformation effect powered by a spring-based smoothing system.
- Real-time visual feedback with natural-looking stretching animation.
- Controller-free interaction using only facial expressions.
- Debug overlay available for landmark visualization.
